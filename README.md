# Cyberdeck

> A documentation-first exploration of laptop hardware architecture through the design and construction of a Linux cyberdeck.

---

## Objectives

- Learn laptop hardware from the motherboard outward.
- Understand how Linux interacts with physical hardware.
- Build a repairable, modular portable computer.
- Document the engineering process from start to finish.
- Create a kid-friendly Linux environment that can evolve over time.

---

## Guiding Principles

- Learn before modifying.
- Document before forgetting.
- Repair before replacing.
- Understand before optimizing.

---

## Planned Topics

- Laptop anatomy
- Motherboard architecture
- CPU & memory
- Storage
- Display subsystem
- Power delivery
- Cooling
- Firmware (UEFI)
- Linux boot process
- Rehousing into a cyberdeck
- Software configuration
